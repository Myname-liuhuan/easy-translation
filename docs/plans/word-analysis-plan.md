# 词性分析功能修改计划

## 需求概述

在翻译软件的输入部分添加词性分析功能：
1. 当用户输入英文单词时，显示该单词的词性和音标
2. 如果是动词/形容词等，显示单词的时态变化
3. 如果用户输入中文且有多个英文翻译，显示所有结果

## 功能设计

### UI 设计

#### 1. 英文单词词性 + 时态提示（方案 B：内联展开式）

在输入框和分隔线之间添加 **词性提示区**：

```
┌──────────────────────────────────────┐
│ [running]                            │
│                                      │
│ ● v. /ˈrʌnɪŋ/  verb 动词    ▼ 时态  │
│   ├── ran (过去式)                    │
│   ├── runs (第三人称单数)             │
│   └── running (现在分词)              │
───────────────────────────────────────│
│ 跑；奔跑                             │
└──────────────────────────────────────┘
```

**设计细节：**
- 词性提示区使用圆点指示器（不同词性不同颜色）
- 音标使用斜体显示
- 词性中英文对照显示
- "时态"点击可展开/收起时态列表
- 背景色比主背景稍浅，形成层次感
- 显示时有淡入 + 上滑动画（staggered reveal）

## 技术方案

### 数据来源

采用 **本地词典优先 + API 备用** 的策略：

#### 1. 本地词典（优先）

使用项目自带的 SQLite 词典数据库 `dict.db`：
- 路径：`dict/dict.db`
- 包含字段：word（主键）、phonetic、translation、pos（词性）、collins、oxford、tag、bnc、frq 等
- 优点：无需网络请求，响应速度快，离线可用
- 缺点：词汇量取决于数据库内容

**查询方式：**
```sql
SELECT word, phonetic, translation, pos, collins, oxford, tag, bnc, frq
FROM dict WHERE word = 'example';
```

#### 2. 外部 API（备选）

当本地词典查不到时，使用 **Free Dictionary API** 作为备选：
- 地址: `https://api.dictionaryapi.dev/api/v2/entries/en/<word>`
- 完全免费，无需 API Key
- 返回数据包含：音标、词性、释义、例句

**备选方案：**
1. CMU Pronouncing Dictionary：包含音标和词性的公开词库

### 实现策略

```
用户输入 → 检测是否为英文单词
         ↓
    本地词典查询 → 找到 → 显示结果
         ↓ 未找到
    Free Dictionary API → 找到 → 显示结果并缓存到本地
         ↓ 未找到
    显示"未找到"提示
```

### 本地词典缓存

首次通过 API 查询到的单词，可写入本地 SQLite 数据库，实现增量更新：

```sql
INSERT OR REPLACE INTO dict (word, phonetic, translation, pos, collins, oxford, tag, bnc, frq)
VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?);
```

### 实现步骤

#### 步骤 1：创建词性分析 composable (`useWordAnalysis.ts`)
- 检测输入是否为单个英文单词
- 调用词典 API 获取词性和音标
- 本地缓存已查询的单词

#### 步骤 2：实现时态显示功能
- 解析 `exchange` 字段获取词形变化
- 添加展开/收起状态管理
- 实现点击复制功能

#### 步骤 3：实现多结果展示功能
- 检测输入语言（中文/英文）
- 本地词典支持模糊查询（LIKE 语句）
- 处理多个翻译结果的数据结构
- 点击选中逻辑

#### 步骤 4：修改 App.vue
- 在输入区和分隔线之间添加词性提示区
- 监听输入变化，触发词性分析
- 添加多结果列表组件

#### 步骤 5：添加样式
- 词性指示器的颜色样式
- 淡入 + 上滑动画效果
- 展开/收起动画
- 列表项 hover 效果
- 适配深色主题

### 数据获取

#### 时态数据来源
- 本地词典 `exchange` 字段：JSON 格式存储词形变化
- Free Dictionary API：`meanings[].inflections[]` 包含时态信息

#### 多结果数据来源
- 本地词典：`SELECT * FROM dict WHERE translation LIKE '%关键词%'`
- 优先按 `collins` 星级排序，再按 `frq` 词频排序
