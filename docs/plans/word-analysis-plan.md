# 词性分析功能修改计划

## 需求概述

在翻译软件的输入部分添加词性分析功能：当用户输入英文单词时，显示该单词的词性和音标。

## 功能设计

### UI 设计

在输入框和分隔线之间添加 **词性提示区**：

```
┌────────────────────────────────┐
│ [running]                      │
│                                │
● v. /ˈrʌnɪŋ/  verb 动词
────────────────────────────────│
│ 跑；奔跑                       │
└────────────────────────────────┘
```

**设计细节：**
- 词性提示区使用圆点指示器（不同词性不同颜色）
- 音标使用斜体显示
- 词性中英文对照显示
- 背景色比主背景稍浅，形成层次感
- 显示时有淡入动画

### 词性颜色映射

| 词性 | 颜色 |
|------|------|
| 名词 (noun) | 紫色 #8B5CF6 |
| 动词 (verb) | 蓝色 #3B82F6 |
| 形容词 (adjective) | 绿色 #10B981 |
| 副词 (adverb) | 橙色 #F59E0B |
| 介词 (preposition) | 灰色 #6B7280 |
| 连词 (conjunction) | 青色 #06B6D4 |
| 代词 (pronoun) | 粉色 #EC4899 |
| 其他 | 白色 #9CA3AF |

## 技术方案

### 数据来源

推荐使用 **Free Dictionary API**：
- 地址: `https://api.dictionaryapi.dev/api/v2/entries/en/<word>`
- 完全免费，无需 API Key
- 返回数据包含：音标、词性、释义、例句

**备选方案：**
1. API + 本地缓存：首次查询后缓存到本地，避免重复请求
2. CMU Pronouncing Dictionary：包含音标和词性的公开词库

### 实现步骤

1. **创建 Vue composable** (`useWordAnalysis.ts`)
   - 检测输入是否为单个英文单词
   - 调用词典 API 获取词性和音标
   - 本地缓存已查询的单词

2. **修改 App.vue**
   - 在输入区和分隔线之间添加词性提示区
   - 监听输入变化，触发词性分析

3. **添加样式**
   - 词性指示器的颜色样式
   - 淡入动画效果
   - 适配深色主题

### 文件修改清单

| 文件 | 修改内容 |
|------|----------|
| `src/composables/useWordAnalysis.ts` | 新增词性分析逻辑 |
| `src/App.vue` | 添加词性提示区 UI |
| `src/dict/` | 如需本地缓存可添加缓存逻辑 |
